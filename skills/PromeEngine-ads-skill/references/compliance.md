# 合规黑名单 + 各国规则

> V_1.0 合规约束分三层:
> 1. Meta 平台通用合规(全市场)
> 2. 各国专项合规(按 D4 自动叠加)
> 3. 业务模式合规(B2B/B2C 不同)

---

## 一、通用黑名单(全市场,headline + primary_text + welcome_message 都禁)

### 1.1 绝对化用语

| 禁 | 替换 |
|---|---|
| 最便宜 / 最低价 / cheapest / lowest price | 具有竞争力的价格 / competitive pricing |
| 最好 / 最佳 / 第一 / best / top / #1 / leading | 优秀的 / excellent / top-tier |
| 唯一 / the only / exclusive | 独特的 / distinctive |
| 完美 / perfect | 卓越的 / outstanding |
| 100% 安全 / 100% safe | 高安全标准 / high safety standards |
| 国家级 / 世界级 / national-grade / world-class | 行业领先 / industry-leading |
| 革命性 / 颠覆性 / revolutionary | 创新的 / innovative |

### 1.2 未求证对比(全禁)

**严禁出现任何"比 X 高/低/快/远 X%"的表述**,即使带 "up to" 软化也禁止。

| ❌ 禁止 | ✅ 替换 |
|---|---|
| "比 Hilux 便宜 12%" | 删除该对比 |
| "续航比 X 多 100km" | "Long range, suitable for daily commute" |
| "省油 15% vs 同级车" | "Fuel-efficient" |
| "Up to 12% more efficient" | "Fuel-efficient (WLTP)" |
| "16% Lower Price" | "Competitive pricing in segment" |

### 1.3 具体补贴金额

| 禁 | 替换 |
|---|---|
| 政府补贴 $5,000 | 享受当地新能源补贴(具体以政府公告为准) |
| EV 退税 30% | 符合资格者可享受 EV 优惠 |
| Free registration | 注册费减免(条件以当地为准) |

理由:补贴金额变化频繁,文案写死会过期且被监管挑刺。

### 1.4 价格绝对值 + FOB 范围(B2B 现货全禁)

**所有价格信息(MSRP / FOB / starting from)全部禁止出现在广告内容中**,价格信息交给客服在 WhatsApp 内沟通。

| ❌ 禁止 | ✅ 处理 |
|---|---|
| "$15,000" / "Starting from $15,000" | 删除,改为 "Contact us for pricing" |
| "FOB Tianjin: $X,XXX-$X,XXX (MOQ ≥10 units)" | 删除,FOB 交给客服聊 |

### 1.5 性能绝对值

| 禁 | 替换 |
|---|---|
| 续航 600km | Long range — up to 600km (WLTP) |
| 0-100 加速 3.5 秒 | High performance — 3.5s 0-100km/h (track conditions) |
| 5.5T 载重 | Heavy duty — payload up to 5.5T |

### 1.6 风险行为画面

- ❌ 超速画面 / 特技漂移 / 违章驾驶
- ❌ 不系安全带 / 儿童在副驾
- ❌ 未授权真人形象 / 明星脸 / 竞品 Logo

### 1.7 货币 / 字符禁忌

- ❌ 严禁 ¥ / RMB / CNY / 人民币符号出现在出口市场素材
- ❌ 严禁中文字符出现在车身 / 车牌 / 标语 / 广告横幅
- ❌ 严禁繁体中文(海外市场)

## 二、各国专项合规(按 D4 国家动态叠加)

> skill 阶段 2 通过 web_search 拉取最新规则,缓存 7 天。
> 以下为常见规则模板,实际执行时以查询结果为准。

### 2.1 中东(AE / SA / EG / IQ / JO)

| 项 | 要求 |
|---|---|
| **AE / SA** | 禁酒精 / 宗教 / 女性身体暴露元素;阿拉伯语母语审核必须 |
| **SA** | 禁周五黄金时段非伊斯兰文化广告;斋月期间夜间投放优先 |
| **价格** | 含 VAT 标注(若涉及价格,但 B2B 现货建议完全不出现价格) |
| **EG / IQ / JO** | 进口许可号要求 + 海关车龄限制 |

### 2.2 非洲(NG / KE / EG / ZA / GH)

| 项 | 要求 |
|---|---|
| **NG 尼日利亚** | 海关车龄限制 ≤ 15 年;需 SONCAP 认证标注 |
| **KE 肯尼亚** | 车龄限制 ≤ 8 年;KEBS 认证 |
| **EG 埃及** | 进口许可 + EAS 认证 |
| 通用 | 英语 / 法语本地化(法语区如 CI/SN) |

### 2.3 拉美(MX / BR / AR / CO / CL / PE)

| 项 | 要求 |
|---|---|
| **MX 墨西哥** | NOM 认证 + 西班牙语本地化 |
| **CL 智利** | 车龄限制 ≤ 3 年(严格)+ INN 认证 |
| **PE 秘鲁** | 进口许可 + 西班牙语 |
| **BR 巴西** | INMETRO 认证 + 葡萄牙语 |
| **CO 哥伦比亚** | RUNT 注册 |
| 通用 | 西/葡语本地化;部分国家有车龄限制 |

### 2.4 中亚独联体(KZ / UZ / RU / GE)

| 项 | 要求 |
|---|---|
| 通用 | **俄语为通用语**(必须本地化) |
| **KZ / RU** | GOST 认证(部分车型) |
| **UZ** | 需进口许可,部分车辆类型受限 |

### 2.5 东南亚(PH / VN / TH / ID / MY)

| 项 | 要求 |
|---|---|
| **PH 菲律宾** | LTO 注册 + 部分右舵市场 |
| **TH 泰国** | 右舵市场 + 皇室相关元素禁用 + 泰文 |
| **VN 越南** | 左舵市场 + 越南语 |
| **ID 印尼** | Halal 认证(若适用)+ Bahasa Indonesia |
| **MY 马来西亚** | 右舵 + 马来语 / 英语 |

### 2.6 欧洲发达(V_1.0 不主投,留作扩展)

| 国家 | 关键要求 |
|---|---|
| DE | 续航必标 WLTP;价格用 "ab" 或 "UVP";WhatsApp GDPR |
| FR | 2022 起所有汽车广告含 3 强制语之一;CO₂ 必标;ARPP |
| UK | ASA CAP Code;续航 WLTP;价格含 OTR |
| 欧盟 | DSA 广告标"为什么投给我";GDPR |

## 三、welcome_message 合规约束

### 3.1 基础要求

- 纯文本第一人称,含产品名 + 开放式问题
- 按目标市场使用当地语言或英语
- **严禁出现具体 FOB 价格 / 报价数字**(让客服在对话内给)
- **字符上限动态**(每次 web_search 拉取 Meta 当前限制,过往撞过 #100 拒绝)

### 3.2 各市场 welcome 示例(B2B 现货)

**英语(IQ / NG / EG / SA / AE 等英语商务市场)**:
> Hi! We're {Brand}'s authorized exporter. Looking forward to your wholesale inquiry — could you share your target market and expected MOQ?

**阿拉伯语(SA / AE / EG / IQ 母语层)**:
> مرحباً! يسعدنا اهتمامك بمنتجاتنا. شاركنا السوق المستهدف وكمية الشراء المطلوبة.

**西班牙语(MX / PE / CO / CL)**:
> ¡Hola! Somos exportadores autorizados de {Brand}. ¿Podrías compartir tu mercado objetivo y la cantidad de compra estimada?

**俄语(KZ / UZ)**:
> Здравствуйте! Мы — авторизованный экспортер {Brand}. Поделитесь, пожалуйста, целевым рынком и планируемым объемом закупки.

## 六、合规更新机制

合规规则会变。建议:
- 每次 skill 阶段 2 用 web_search 拉最新(政策类 <3 个月时效)
- 缓存 7 天到内部 `platform_country_rules`
- 用户在同会话改了目标平台/国家 → 立即重新查
