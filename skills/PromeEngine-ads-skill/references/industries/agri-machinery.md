## 二、business_mode: b2b-general(B2B 一般批发)主力

### 2.1 创意主线(预设)

**「专业可靠」** — 围绕"农场设备品质 / 售后保障 / 长期合作"展开。

### 2.2 卖点话术(参考行业通用)

**优先词**:
- 拖拉机:马力 / horsepower / 燃油经济 / fuel-efficient / 多用途 / multi-purpose
- 收割机:产能 / capacity / 收割效率 / 节省人工
- 通用:OEM 支持 / 售后保障 / 配件供应 / spare parts / 经销网络

**禁忌词**:试驾 / 到店 / 家用(C 端术语)

### 2.3 漏斗结构(参考 b2b-general)

```
广告 → 点击 → WhatsApp(询设备规格 / 报价)
     → 高价值判定(机型 + 数量 + 农场规模)
     → 报价 + OEM 资料
     → 合作(发合同)
     → 成交
```

## 三、business_mode: b2b-spot(B2B 现货批发)

### 3.1 创意主线(预设)

**「现货可得」** — 类似汽车 b2b-spot,但场景换成农场设备。

### 3.2 图片策略

参考 automotive.md b2b-spot 12 类清单,**视觉要素替换为农机场景**:
- 堆场 → 农机仓库
- 单车主图 → 单机主图
- 落地场景 → 田间作业

### 3.3 受众词调整

**优先**:
- farmer / commercial farmer / large-scale farm
- agricultural cooperative / 农业合作社
- agri-equipment dealer / 农机经销商
- importer / agri-machinery importer
