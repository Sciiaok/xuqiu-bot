## 二、business_mode: b2b-general(B2B 大项目)主力

### 2.1 创意主线(预设)

**「专业项目伙伴」** — 围绕"项目方案 / 长期运维 / 大规模交付"展开。

### 2.2 卖点话术(参考行业通用)

**优先词**:
- 项目方案 / project solution / EPC support
- 技术参数:转换效率 / efficiency / 衰减率 / degradation / IP 等级
- 售后:质保 / warranty / O&M support / 25 年发电保证
- 渠道:经销商 / 安装商 / dealer / installer
- 单 W 价格(BOM):仅作内部参考,广告不出现

**禁忌词**:试驾 / 到店 / 家用(后者仅 b2c 户用用)

### 2.3 漏斗结构

```
广告 → 点击 → WhatsApp(询项目可行性)
     → 高价值判定(项目规模 + 国家 + 时间窗)
     → 项目方案 + 报价
     → 框架签约
     → 项目交付 / 长期供货
```

## 五、通用合规

### 5.1 各市场认证要求

| 市场 | 认证 |
|---|---|
| 欧洲 | CE / TUV / IEC / VDE / EN 标准 |
| 北美 | UL 1741 / IEEE 1547 / NEC |
| 澳洲 | CEC / AS/NZS |
| 中东 | SASO(SA)/ ESMA(AE) |
| 巴西 | INMETRO |
| 东南亚 | 各国当地认证 |

### 5.2 营销合规

- 严禁夸大发电量(必须标"理论值 / 实际取决于安装条件")
- 严禁绝对化用语
- 价格不进广告(让客服在 WhatsApp 沟通)
