## 二、business_mode: b2b-spot(B2B 现货批发)★ 主力业务

### 2.1 创意主线

**「现货可信」** — 所有素材围绕"真有车、能即买即发、单证齐全"展开。

支撑信息:
- 真实仓储照(数十台车成排)
- 库存数字 + 发运周期
- 物流/单证可视化
- 经销商口吻的 welcome

### 2.2 卖点话术 [图策 + 20天]

**优先使用词**:
现货 / In Stock / Ready to Ship / MOQ / FOB / CIF / OEM 支持 / 检测报告 / 出口许可 / 货运保险 / 经销资源 / 批量供货 / 即买即发

**严禁使用词**(消费者向):
试驾 / 到店 / 家用 / 续航焦虑 / 智能化 / 充电桩 / 个人车主

**热门型号**(20 天询单高频):
Geely Galaxy M9 / Geely Starshine 6 / Geely Xingyao 8 / Foton AUMARK / Foton Auman / Foton Tunland V9 / Foton Galaxy 9

### 2.3 漏斗结构 6 节点 [投同]

```
广告曝光 → 点击 → WhatsApp 对话(询现货)
       → 高价值判断(车型 + 数量 + 目的港)
       → 报价单 / OEM 资料
       → 合作(发合同)
       → 成交
```

| 节点 | 客服 SLA | KPI |
|---|---|---|
| 广告曝光 | -- | CPM、覆盖经销商人数 |
| 点击 | -- | CTR、CPC |
| WhatsApp 首响 | AI ≤ 1 分钟 / 人工 ≤ 2 小时 | Click → 首响率 |
| 高价值判断 | ≤ 24 小时 | Conversation → Qualified Lead |
| 报价单 | ≤ 24-48 小时 | CPQL |
| 合作 | ≤ 48-72 小时 | 签约率 |
| 成交 | -- | 总转化周期 30-60 天 |

**关键约束**(投放同事):
- 转化周期 30-60 天(广告 → 实际成交)
- 单线索价值数万美元,CPQL $30-50 行业上限可接受
- ★ 您 20 天实测 CPQL = **$14.49(严)/ $2.17(宽)**,远低于上限,有充足扩量空间

### 2.4 12 类图片清单 [图策]

> 完整保留用户图片策略文档第二部分。每类含适用场景 + 视觉要求 + Prompt 骨架 + 数据支撑。

#### #1 现车堆场全景图(主力冷启动)

- **适用场景**:冷量级开发新经销商 / 进口商,首次建立"真有现货"信任
- **视觉要求**:真实实拍(非渲染);数十台车成排;白天自然光;品牌横幅;车牌/VIN 打码;一张图说清"车在场"
- **Prompt 骨架**:
```
Real photography (not rendered) of {N}+ same-model {车型} parked in rows at port-bonded warehouse, daytime natural light, "{Brand} In Stock" banner visible, license plates blurred or blank, no Chinese characters, wide-angle composition showing scale, dust-free clean environment, full-frame composition (no empty corners), 3/4 front three-quarter view angle of the cars in foreground
```
- **数据支撑**:静态图 CTR 0.8%-1.5%;真实/UGC 风格压低 CPM;您 20 天 Android 设备 CTR 3.08%、Foton 占多个头部 ad_set

#### #2 单车 3/4 前侧主图(转化主力)

- **适用场景**:单一爆款车型 campaign、再营销暖受众
- **视觉要求**:3/4 前侧角(广告最具吸引力角度);干净/中性背景;车型名;无中文字
- **Prompt 骨架**:
```
{车型} front three-quarter view, clean neutral background, professional product photography style, brand identity visible on the front grille, license plate area blank or blurred, no Chinese characters, soft studio lighting
```
- **数据支撑**:产品化主体图与 Meta Feed 版位天然契合

#### #3 库存矩阵网格图

- **适用场景**:经销商招募、展示全系/可选范围
- **视觉要求**:4-8 车型缩略宫格;每格"In stock: NN"库存徽章;不挂价格;统一品牌框
- **Prompt 骨架**:
```
Grid layout (2x3 or 3x3) showing {N} car models, each cell with model name and "In Stock: NN" badge in green color, unified brand frame, no price tags, clean product photography style, white or light grey background
```

#### #4 库存数据卡 / 现货徽章

- **适用场景**:暖受众催单、稀缺感营造
- **视觉要求**:超大数字 + "READY TO SHIP" + 发运周期(用区间);货币中性;无绝对价
- **Prompt 骨架**:
```
Bold infographic poster, large "READY TO SHIP" stamp at top, huge number "{N} units in stock" centered, shipping range "14-21 days delivery" below, currency-neutral design (no $ signs), brand colors, modern flat design style
```

#### #5 集装箱 / 装船物流图

- **适用场景**:打消交付顾虑,面向 FOB / CIF 进口商
- **视觉要求**:汽车装柜或滚装上船;港口吊机/船;"FOB/CIF"轻标注;不承诺精确到天
- **Prompt 骨架**:
```
Cars being loaded into shipping container or driven up Ro-Ro vessel ramp at busy port, gantry crane and cargo ship in background, "FOB / CIF available" small text overlay at corner, dawn or daytime lighting, no specific delivery date commitments
```
- **数据支撑**:B2B 买家怕"库存不清+延误";物流可视化降低点击后摩擦

#### #6 WhatsApp 对话引导图(全 campaign 标配元素)

- **适用场景**:所有 CTW 广告,明示"点击 = 进入对话"

#### #7 真人销售 / 团队出镜图

- **适用场景**:高客单 B2B、陌生市场建信任

### 2.7 衔接结构性规则(3 条强约束) [图策]

1. **承诺-兑现一致性**:堆场图引来的对话,首响必须先确认"有现货",而不是先发产品手册。否则点击-对话落差被算法识别为低质,推高 cost per conversation
2. **图片决定话术分支**:同一 WhatsApp 号下,不同素材应配不同 welcome_message 与 icebreaker,让首条对话直接进入对应资质判定,缩短到"热询盘"的路径

## 五、通用合规(所有 business_mode 共用) [V4 + 公开法规]

### 5.1 Meta 平台通用合规(全市场)

**禁止**:
- 绝对化用语:最佳 / 第一 / 唯一 / 完美 / 100% 安全
- 未求证性能宣称(续航/加速/安全)
- 误导性对比 / 鼓励危险驾驶画面
- 未授权真人形象 / 品牌标识

**必须**:
- 性能参数标注测试条件(WLTP / EPA / NEDC / CLTC)
- 价格相关标注"起售价"等限定语
- 车牌不出现中文字
- 货币使用目标市场货币(严禁 ¥/RMB/CNY)

### 5.2 各市场专项合规(完整查询见 references/compliance.md)

| 市场 | 关键要求 |
